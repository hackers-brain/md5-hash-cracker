# MD5 Hash Cracker
MD5 Hash Cracker Tool by HackersBrain
## Version
**Hash Cracker V1.0**

## Requirements
**Python3**

## Installation
Clone or Download this Repository
```
git clone https://github.com/hackers-brain/md5-hash-cracker.git
```
Change Directory
```
cd md5-hash-cracker
```
Run the .py file
```
python3 md5_hash_cracker.py
```
